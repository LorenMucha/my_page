Climbing Diary

Oct 1, 2020 android
Code on GitHub

Übersicht

Das Projekt habe ich aus dem Bedürfnis heraus entwickelt, eine eigene Möglichkeit zu besitzen, die von mir gepunkteten Touren abzuspeichern und auszuwerten. Hiermit ist ein eigenes Projekt entstanden.

Funktionen

  • erstellen, aktualisieren und löschen von Touren und Projekten
  • Backup und Wiederherstellung der zugrundeliegenden Datenbank
  • statistische Auswertung der erbrachten Leistung als
    • Balkendiagramm
    • Liniendiagramm
    • Tabelle
  • Filter und Suchfunktion
  • Autovervollständigung zur schnellen Eingabe neuer Routen/Projekte
  • durch den Einsatz des Observer Pattern aktualisieren sich alle Sichten, bei einer Manipulation der Daten.
